# latam-food-data

Projeto de engenharia de dados para uma indústria alimentícia fictícia com atuação na LATAM, utilizando Databricks, PySpark, integração com APIs e versionamento no GitHub.

## Objetivo
Construir um pipeline de dados para coletar, transformar e disponibilizar informações analíticas de parceiros, produtos, vendas e indicadores estratégicos.

## Tecnologias utilizadas
- Databricks
- PySpark
- Python
- APIs REST
- GitHub
- Delta Lake

## Etapas do projeto
1. Coleta de dados via API
2. Tratamento e transformação com PySpark
3. Armazenamento em camadas
4. Análise exploratória e geração de indicadores
5. Versionamento e documentação no GitHub

## Estrutura inicial
- `README.md`: documentação do projeto
- notebooks Databricks: desenvolvimento das rotinas
- scripts futuros: integrações e transformações adicionais
